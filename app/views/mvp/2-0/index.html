Version notes

  • Iterations are based on feedback from Round 1 of UR, conducted on version 1 of MVP
  • Mockup of award notice letter - Round 2 version has been iterated to:
    • use a gender neutral name (Alex);
    • include the short URL that will be used for the service, confirmed by GOV.UK team;
    • include a line about more changes coming in future, as some users in round 1 thought bank change was all they would ever be able to do
  • JIRA tickets for prototype iterations
  • Updated ID&V journey to skip new credential confusion page they asked us to test and also skip the registration process with users to keep sessions to time
  • Added full name to SP overview page so it's clear whose account user has logged into
  • Added marital status into overview page, as 2 users mentioned seeing they needed to report a change to their marital status in the letter, but couldn’t see that info in the overview page
  • Moved payment frequency to top of payment schedule section to break the link for some users between amount and frequency
  • Looking into playing back correspondence address if different from residential - feedback from cross-retirement leads, code in but hidden for UR
  • Content updated in check answers page when roll number not provided - feedback from cross-retirement leads
  • Added additional info on overview screen that more changes are coming to online service (links with letter content) as some users were concerned the service would only ever cater to bank changes
